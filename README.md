# Fruits-Retail-Sales-Analytics-Dashboard-Power-BI

Project Overview

This project is an interactive Power BI dashboard built to analyze retail fruit sales data across multiple stores, customers, brands, and product categories.

The dashboard provides a complete view of sales performance, customer trends, store ranking, and time-based analysis to support data-driven business decisions.
